# 艺术素养培训机构排课系统

## 项目概述
一个专为艺术培训机构设计的智能排课系统，支持音乐、舞蹈、美术三大课程类型。

## 技术栈
- Vue 3 + Vite
- Naive UI (现代化UI框架)
- Pinia (状态管理)
- Vue Router (路由)
- Day.js (日期处理)

## 用户角色

### 1. 学生/家长
- 查看个人课表
- 查看课程详情
- 查看调课通知
- 查看上课记录

### 2. 代课老师
- 查看个人课表
- 提交调课申请
- 查看学生名单
- 记录上课情况

### 3. 教务管理员
- 一键智能排课
- 手动排课/调课
- 管理教师信息
- 管理学生信息
- 管理教室信息
- 管理课程信息
- 查看所有课表
- 数据统计分析

## 核心功能

### 1. 智能排课
- **按班级排课**: 为整个班级安排固定时间的课程
- **按教室排课**: 充分利用教室资源，自动分配课程
- **按教师排课**: 根据教师空闲时间自动安排课程
- **冲突检测**: 
  - 教师时间冲突检测
  - 学生时间冲突检测
  - 教室占用冲突检测
- **灵活时长**: 支持45分钟、60分钟、90分钟、120分钟等多种时长

### 2. 课表管理
- **多维度查看**:
  - 周课表（7天视图）
  - 日课表（单日详细）
  - 教师课表
  - 学生课表
  - 教室课表
  - 班级课表
- **可视化展示**: 时间轴+卡片式布局
- **打印功能**: 支持导出PDF和打印

### 3. 调课管理
- **临时调课**: 因突发情况调整课程时间
- **课程取消**: 取消某节课程
- **课程补课**: 为取消的课程安排补课
- **调课审批**: 教师申请，管理员审批
- **调课通知**: 自动通知相关学生和教师

### 4. 课程类型

#### 音乐课程
- 钢琴（一对一/小组课）
- 声乐（一对一/小组课）
- 吉他（一对一/小组课）
- 古筝（一对一/小组课）
- 小提琴（一对一/小组课）

#### 舞蹈课程
- 芭蕾舞（小组课）
- 中国舞（小组课）
- 街舞（小组课）
- 拉丁舞（小组课）

#### 美术课程
- 素描（小组课）
- 水彩（小组课）
- 国画（小组课）
- 创意美术（小组课）

### 5. 基础数据管理

#### 教师管理
- 教师信息（姓名、联系方式、专业）
- 可授课程
- 工作时间设置
- 请假管理

#### 学生管理
- 学生信息（姓名、年龄、联系方式）
- 报名课程
- 上课记录
- 家长信息

#### 教室管理
- 教室信息（名称、容量、位置）
- 教室类型（音乐教室、舞蹈教室、美术教室）
- 设备配置

#### 班级管理
- 班级信息（名称、课程类型、人数）
- 学生名单
- 授课教师
- 上课时间

## 页面结构

```
/login                    - 登录页
/dashboard                - 仪表盘（根据角色显示不同内容）

# 排课管理（管理员）
/schedule
  /auto                   - 智能排课
  /manual                 - 手动排课
  /adjust                 - 调课管理
  /conflict               - 冲突检测

# 课表查看
/timetable
  /week                   - 周课表
  /teacher/:id            - 教师课表
  /student/:id            - 学生课表
  /classroom/:id          - 教室课表
  /class/:id              - 班级课表

# 基础数据（管理员）
/teacher
  /list                   - 教师列表
  /add                    - 添加教师
  /edit/:id               - 编辑教师

/student
  /list                   - 学生列表
  /add                    - 添加学生
  /edit/:id               - 编辑学生

/classroom
  /list                   - 教室列表
  /add                    - 添加教室
  /edit/:id               - 编辑教室

/class
  /list                   - 班级列表
  /add                    - 添加班级
  /edit/:id               - 编辑班级

/course
  /list                   - 课程列表
  /add                    - 添加课程
  /edit/:id               - 编辑课程

# 个人中心
/profile                  - 个人信息
/settings                 - 系统设置
```

## 设计风格

### 主题色
- 主色调: 紫色系 (#7C3AED, #8B5CF6, #A78BFA)
- 辅助色: 粉色系 (#EC4899, #F472B6)
- 成功色: #10B981
- 警告色: #F59E0B
- 错误色: #EF4444

### 设计特点
- 现代化卡片式布局
- 柔和的圆角和阴影
- 流畅的动画过渡
- 艺术感的图标和插图
- 响应式设计，支持移动端
